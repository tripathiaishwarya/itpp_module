crc_encode

CRC Encoding

Calling Sequence

a = crc_encode(b,c)

Parameters

a:

output single dimension binary encoded array of {0,1}.

b:

input single dimension binary array of {0,1}.

c:

input string.

Description

CRC Encoding.

The function encodes the input bit stream using the CRC Encoding scheme

It is expected that the user inputs an encoded binary array of (0,1) and string(defining the CRC encoding scheme) while calling the crc_encode() function and expect the function to return a binary array of (0,1) as decoded output in scilab console.

Examples

b=[1 0 0 1 1 0];
c='CRC-4';
a=crc_encode(b,c)
