psk_mod

PSK modulation

Calling Sequence

a = psk_mod(b,c)

Parameters

a:

output single dimension complex array.

b:

input single dimension array of {0,1}.

c:

input integer that should be a power of 2.

Description

M-ary PSK modulator.

This function implements the M-ary PSK modulator with constellation points, where . The symbol numbering is counter clockwise starting from the real axis, i.e. symbol . The bitmap is Gray encoded. The symbol energy is normalized to 1.

It is expected that the user inputs a single dimension binary matrix of (0,1) and an integer m(power of 2) in scilab engine while calling the itpp_bpsk_mod() function and expect the function to return a single dimension complex matrix as output in scilab console.

Examples

b=[1 0 0 1 1 1];
c=4;
a=psk_mod(b,c)
