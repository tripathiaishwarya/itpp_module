itpp_block_interleaver

Function for block interleaving. Adds some zeros.

Calling Sequence

Vec< T > block_interleave(int in_rows, int in_cols, const Vec< T > &input)

Arguments

  • in_rows - Set the number of rows for block interleaving
  • in_cols - Set the number of columns for block interleaving
  • input - Input Vector

Description

Block Interleaver Class. Data is written row-wise and read column-wise when interleaving.

Examples

a = [1 -1 1 1]

b = block_interleave(2,2,a)

b = [1 1 -1 1]
