berc_get_errorrate

Bit Error Rate Counter (BERC) Class

Calling Sequence

BERC berc(int indelay=0, int inignorefirst=0, int inignorelast=0);
berc.count(const bvec &in1, const bvec &in2)
result = berc.get_errors();

Arguments

  • indelay - delay is positive if in2 is a delayed replica of in1 and negative otherwise.
  • inignorefirst - Used if errors in the begining are to be ignored.
  • inignorelast - Used if errors in the end are to be ignored.
  • in1 - Input signal 1
  • in2 - Input signal 2
  • Result - Returns estimated bit error rate between in1 and in2

Description

Bit Error Rate Counter (BERC) Class

Examples

in1 = [1 0 1 1]
in2 = [1 1 1 1]
result = berc_get_errorrate(0, 0, 0, in1, in2)
result = 0.25
