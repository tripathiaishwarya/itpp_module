rc_shape_symbols

Raised Cosine (RC) Pulse Shaper.

Calling Sequence

Vec = itpp_shape_symbols(double roll_off, int filter_length=6, int upsampling_factor=8, const Vec< T1 > &input)

Arguments

  • roll_off - Roll-Off Factor. Must be between 0 and 1.
  • filter_length - Filter Length. Must be even.
  • upsampling_factor - Upsampling Factor
  • input - Input symbols to shape
  • Vec - Shaped pulse

Description

Shape the input symbols performing upsampling.
