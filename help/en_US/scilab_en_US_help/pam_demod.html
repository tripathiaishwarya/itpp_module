pam_demod

M-ary PAM de-modulator with real symbols

Calling Sequence

BitVec = pam_demod(int M, const vec &signal)

Arguments

  • M - Set the size of the signal constellation
  • signal - Input signal
  • BitVec - De-Modulated Bit Vector returned

Description

The PAM class implements an M-ary PAM de-modulator with the following signal values: Symbol numbering is from right to left in the increasing order. The Gray encoding of bits to symbols is used. The constellation symbols are normalized so that the average energy is equal to 1. That is, normalized with

M must be some square of 2

Examples

vec = [-1 1 -1 -1]
pam_demod(2, vec)
output = [1 0 1 1]
