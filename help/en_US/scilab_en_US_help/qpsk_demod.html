qpsk_demod

QPSK demodulation

Calling Sequence

a = qpsk_demod(b)

Parameters

a:

output single dimension array of {0,1}.

b:

input single dimension complex array.

Description

QPSK demodulator. This is a special version of the PSK demodulator with constellation points. Symbol numbering is counter clockwise starting from the real axis. Bits are Gray coded onto symbols. Symbol energy is normalized to 1.

It is expected that the user inputs a single dimension complex matrix in scilab engine while calling the qpsk_demod() and expect the function to return a single dimension binary matrix of {0,1} as output in scilab console.

Examples

b=[1 %i -%i -1];
a=qpsk_demod(b)
