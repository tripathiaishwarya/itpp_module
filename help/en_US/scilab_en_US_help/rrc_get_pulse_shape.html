rrc_get_pulse_shape

Root Raised Cosine get pulse shape

Calling Sequence

a = rrc_get_pulse_shape(b,c,d)

Parameters

a:

Output pulse shape vector.

b:

Roll off factor(non zero).

c:

integer filter length.

d:

integer upsampling factor.

Description

Root Raised Cosine get pulse shape

The function get_pulse_shape() gives the shape of the pulse which will be further used in upsampling and shaping of the input signal

It is expected that the user inputs roll off factor, filter length and upsampling factor in scilab engine while calling the rrc_get_pulse_shape() function and expect the function to return pulse shape vector as output in scilab console.

Examples

b=0.5;
c=6;
d=8;
a=rrc_get_pulse_shape(b,c,d)
