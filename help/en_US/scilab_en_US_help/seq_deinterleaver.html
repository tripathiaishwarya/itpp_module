seq_deinterleaver

Sequence Deinterleaver

Calling Sequence

a = seq_deinterleaver(b,c)

Parameters

a:

output deinterleaved vector.

b:

array of input interleaved symbols.

c:

interleaver depth.

Description

Sequence Deinterleaver.

This function implements Sequence Deinterleaving on input bit stream

It is expected that the user inputs a array of input interleaved symbols and an integer interleaver_depth in scilab engine while calling the seq_deinterleaver() function and expect the function to return the deinterleaved vector as output in scilab console.

Examples

b=[3 4 1 2 7 8 5 6];
c=4;
a=seq_deinterleaver(b,c)
