rrc_shape_symbols

Root Raised Cosine shape symbols

Calling Sequence

a = rrc_shape_symbols(b,c,d,e)

Parameters

a:

Output pulse shape vector.

b:

Input symbols vec .

c:

Roll off factor(non zero).

d:

integer filter length.

e:

integer upsampling factor.

Description

Root Raised Cosine shape symbols

The function shape_symbols() shapes the the input symbols performing upsampling.

It is expected that the user inputs input symbols vector, roll off factor, filter length and upsampling factor in scilab engine while calling the rrc_shape_symbols() function and expect the function to return pulse shape vector as output in scilab console.

Examples

b=[-1 1 -1 1 -1];
c=0.5;
d=6;
e=8
a=rrc_shape_symbols(b,c,d,e)
