BSC

Binary Symmetric Channel

Calling Sequence

outputdata = bsc(data,p)

Arguments

  • data - Input signal data
  • p - Probability of bit error

Description

ndata = bsc(data,p) passes the binary input signal data through a binary symmetric channel with error probability p. The channel introduces a bit error with probability p, processing each element of data independently.

Examples

bsc([1 0 1 1],0.3)
ans = [1 0 1 1]
