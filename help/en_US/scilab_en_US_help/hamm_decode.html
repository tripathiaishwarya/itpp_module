hamm_decode

Hamming Decoding

Calling Sequence

a = hamm_decode(b,c)

Parameters

a:

output single dimension binary decoded array of {0,1}.

b:

input single dimension binary encoded array of {0,1}.

c:

input positive non zero integer.

Description

Hamming Decoding.

The function decodes the encoded input bit stream using the Hamming Decoding scheme

It is expected that the user inputs an encoded binary array of (0,1) and an integer m while calling the itpp_hamm_encode() function and expect the function to return a binary array of (0,1) as decoded output in scilab console.

Examples

b=[0 0 1 0 0 1 1 1 0 1 1 0 1 0];
c=3;
a=hamm_decode(b,c)
