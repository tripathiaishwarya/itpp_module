hamm_encode

Hamming Encoding

Calling Sequence

a = hamm_encode(b,c)

Parameters

a:

output single dimension binary encoded array of {0,1}.

b:

input single dimension binary array of {0,1}.

c:

input positive non zero integer.

Description

Hamming Encoding.

The function encodes the input bit stream using the Hamming Encoding scheme

It is expected that the user inputs a single dimension binary matrix of (0,1) and an integer m while calling the hamm_encode() function and expect the function to return a single dimension binary matrix of (0,1) as encoded output in scilab console.

Examples

b=[0 0 1 1 1 0 1 0];
c=3;
a=hamm_encode(b,c)
