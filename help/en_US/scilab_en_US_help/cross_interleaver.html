cross_interleaver

Function for cross interleaving. Adds some zeros.

Calling Sequence

Vec < T > cross_interleaver(int in_order ,const Vec< T > &input)

Arguments

  • in_order - Used to set the order of the Cross Interleaver
  • input - Input Vector

Description

Cross Interleaver

Examples

a = [1 -1 1 1]

b = cross_interleaver(2,a)

b = [1 0 1 -1 0 1 0 0]
