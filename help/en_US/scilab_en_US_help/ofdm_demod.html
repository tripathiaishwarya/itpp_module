ofdm_demod

Class for modulating and demodulation of OFDM signals using the FFT.

Calling Sequence

Output = ofdm_demod(int inNfft, int inNcp, int inNupsample=1, const cvec &input)

Arguments

  • Nfft - The size of the FFT. Nfft >= 2
  • Ncp - The length of the cyclic prefix. Ncp must be >=0 and <=Nfft
  • Nupsample - The upsampling factor (default=1)
  • input - Modulate complex data symbols. Length of input must be an integer multiple of Nfft+Ncp.

Description

Class for modulating and demodulation of OFDM signals using the FFT. The modulated signal is normalized taking into account the cyclic prefix.

Examples

input = [1.1547005  1.1547005i  1.1547005  1.1547005  -1.1547005i  1.1547005]
output = ofdm_demod(2,1,1,a)
output = [1+i  -1+i  1-i  -1-i]
